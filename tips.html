Tips for calculator

  • Use 0 for false and 1 for true.
  • Use jmp for if else statements

    0 or 1
    3
    *
    jmp ]0
    <run if 0>
    jmp 2
    <run if 1>

  • For the numerator of a Rational use:

    3
    5
    /
    >> 0
    den
    *

  • To copy the lowest element

    #
    >> ]0
    del 1
